# Icon Migration: Emojis to Heroicons

## Plan
Replace all emoji icons with Heroicons throughout the application.

## Components to Update
1. Layout.jsx - Navigation sidebar icons
2. Dashboard.jsx - Stat cards, quick actions, welcome section
3. Documents.jsx - Document type selection, recent documents list
4. Issuances.jsx - Filter buttons, table type column
5. AnimatedCounter.jsx - Stat card icon wrapper
